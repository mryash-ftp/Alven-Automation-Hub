# 1) ⚡ PyConvert Pro
Simple GUI tool to convert Python scripts (.py) into standalone Executables (.exe).

### 🛠️ Requirements
* Python 3.x
* `pyinstaller` (Script will auto-install if missing)

### 🚀 How to Use
1. Run `PyConvert-Pro.py`.
2. Browse and select your `.py` file.
3. (Optional) Select an `.ico` file for the icon.
4. Click **"CONVERT TO EXE"** and wait for the success message.

# 2) 📂 PyGen Flux
Automated bulk file and folder creator with custom naming and date stamping.

### 🛠️ Requirements
* Python 3.x
* No external libraries needed (Uses standard OS & Tkinter).

### 🚀 How to Use
1. Run `Pygen Flux.py`.
2. Select the **Location** where you want to create folders/files.
3. Enter the **Name** and **Quantity**.
4. Choose type (Folder/Python/Text) and click **"GENERATE"**.
