#  📂 PyGen Flux
Automated bulk file and folder creator with custom naming and date stamping.

### 🛠️ Requirements
* Python 3.x
* No external libraries needed (Uses standard OS & Tkinter).

### 🚀 How to Use
1. Run `Pygen Flux.py`.
2. Select the **Location** where you want to create folders/files.
3. Enter the **Name** and **Quantity**.
4. Choose type (Folder/Python/Text) and click **"GENERATE"**.
